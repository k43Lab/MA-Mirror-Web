Github 加速
什么是 Github 加速？
这是一项对Github文件以及api进行国内加速的服务，主要针对解决国内无法访问Github问题。
使用
如果您要对https://github.com/RecursiveG/UniSkinMod/releases/download/1.4-dev2/UniSkinMod-1.9-forge1865-1.4-dev2.jar，那么就在这个地址前面加上https://proxy-gh.1l1.icu/字样，即https://proxy-gh.1l1.icu/https://github.com/RecursiveG/UniSkinMod/releases/download/1.4-dev2/UniSkinMod-1.9-forge1865-1.4-dev2.jar，对于api的加速操作和这个一样。

要求
我们严禁对利用Github仓库作为图床，或加速私人字体，私人图片视频媒体资源文件等进行加速，我们会定期严查这类资源，并进行封禁。
Waring 注意
这只是对文件以及API加速，并不是所谓的Github镜像站。
